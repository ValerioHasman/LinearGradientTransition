Transição de Gradiente Linear

Uma classe JavaScript que possibilita fazer transição em gradiente uma vez que o CSS não consegue fazer com transition e linear-gradient.

Usabilidade e demosntração

Podendo ter sua aplicação em qualquer lugar no DOM.

Usando esperarDeRodar()
Usando idaEVolta
Usando ehUmLoop

Instanciando o objeto

A seguir temos um exemplo de instância do objeto por completo. Para a instância do objeto é apenas obrigatório dar valor ao “alvo”, ao “inicio” e ao “final”, e para que aconteça a execução o método “executar” deve ser acionado.

Atributo Tipo Valor de exemplo Descrição
inicio string linear-gradient(45deg, rgba(144,98,61,0) 0%, rgba(235,141,73,0) 100%) Começo do gradiente, necessita da string completa.
final string linear-gradient(276deg, rgba(235,141,73,1) 0%, rgba(144,98,61,1) 100%) Define o fim do gradiente, necessita da string completa.
segundos number 4 Determina o tempo de transição da animação.
idaEVolta boolean false Faz com que ao terminar a animação ele possa executar uma nova animação começando pelo fim e terminando no começo repetitivamente.
ehUmLoop boolean false Faz com que ao terminar a animação execute desde o começo repetitivamente.
interruptor boolean false Finalize a animação imediatamente.
executar() function Método que inicializa o processo de transição.
esperarDeRodar() function function Método que esperar de rodar onde um evento de finalização é acionado.
